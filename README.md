# 🚀 FlowTask - Modern Todo App

FlowTask is a modern and user-friendly task management application designed to help users organize, track, and complete their daily tasks efficiently. With an intuitive interface and productivity-focused features, FlowTask simplifies task management and boosts productivity.

## 🌟 Features

### ✅ Task Management

* Create new tasks instantly
* Edit existing tasks
* Delete tasks when no longer needed
* Mark tasks as completed
* Undo completed tasks

### 📋 Task Organization

* Categorize tasks
* Prioritize tasks (High, Medium, Low)
* Filter tasks based on status
* Search tasks quickly

### 🎯 Productivity Features

* Track completed and pending tasks
* Real-time task updates
* Progress monitoring
* Clean and distraction-free interface

### 💾 Data Persistence

* Save tasks locally
* Automatic data storage
* Tasks remain available after page refresh

### 🎨 Modern User Interface

* Responsive design
* Mobile-friendly layout
* Smooth animations and transitions
* Modern and attractive user experience

## 🛠️ Tech Stack

### Frontend

* HTML5

### Additional Technologies

* Local Storage API
* Responsive Design Principles
