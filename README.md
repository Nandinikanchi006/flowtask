# 🚀 FlowTask – Modern Todo App

FlowTask is a modern and intuitive task management application designed to help users organize their daily activities, track progress, and improve productivity. The application provides a clean user interface, responsive design, and essential task management features for an efficient workflow.

---

## 📌 Overview

Managing tasks effectively is essential for productivity. FlowTask simplifies this process by allowing users to create, organize, update, and track tasks in a user-friendly environment.

Whether you're a student managing assignments, a professional tracking work tasks, or someone organizing daily activities, FlowTask helps you stay focused and productive.

---

## ✨ Features

### ✅ Task Management

* Add new tasks instantly
* Edit existing tasks
* Delete unwanted tasks
* Mark tasks as completed
* Track pending tasks

### 📊 Productivity Tracking

* View completed tasks
* Monitor pending tasks
* Track overall progress
* Improve daily productivity

### 🔍 Task Organization

* Organize tasks efficiently
* Quick task access
* Easy task updates

### 💾 Data Persistence

* Tasks are stored locally
* Data remains available after page refresh
* Automatic saving using browser storage

### 🎨 Modern UI/UX

* Clean and responsive design
* User-friendly interface
* Smooth interactions
* Mobile-friendly layout
